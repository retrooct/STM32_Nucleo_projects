
## What is this project:
A bare-metal retro game console built on the STM32L476 using C and ARM assembly. The project implements a custom display driver, framebuffer graphics, controller input, and game logic without an operating system or graphics libraries, showcasing low-level embedded systems programming. Akin to a video game console processing a hard-coded software.


### This project is designed to demonstrate:

* Register-level peripheral control
* Memory-mapped I/O
* Bare-metal C programming
* ARM assembly integration
* I2C communication
* Display-driver development
* Framebuffer management
* Bitwise pixel manipulation
* GPIO-based controller input
* Timer-based game loops
* Interrupt handling
* Basic sprite and tile rendering
* Resource management on a microcontroller
* Finite-state-machine-based game logic
